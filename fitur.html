Mengapa Sephon 32?

Lebih dari Sekadar Kit Drone Biasa

Real-Time Dashboard

Data sensor dikirim secara live ke dashboard monitoring via Wi-Fi ESP32. Lihat perubahan kualitas air seketika.

Wi-Fi Terintegrasi

Koneksi nirkabel langsung dari ESP32 ke smartphone atau PC. Tidak perlu hardware tambahan.

Open Source Code

Semua source code tersedia dan siap dikembangkan. Modifikasi sesuai kebutuhan proyek kamu.

Dokumentasi Lengkap

Panduan step-by-step dari perakitan hardware hingga pemrograman ESP32 dan pengolahan data sensor.

Desain Semi-Underwater

Rangka drone tahan air dirancang khusus agar sensor dapat bekerja optimal di dalam dan permukaan air.

Modular & Expandable

Tambahkan sensor lain sesuai kebutuhan. Arsitektur modular memudahkan pengembangan fitur baru.
